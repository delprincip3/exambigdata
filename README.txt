## Istruzioni

1. Inserire il proprio nome e cognome nel file nome_cognome.txt

## MariaDB o MySQL o PostgreSQL
2. Produrre uno script che esegua il load del CSV in esercizi/input_5000.csv
3. Lo script, tramite query, dovrà restituire il numero delle email che terminano con ".com"

## MongoDB
4. Produrre uno script che esegua il load del CSV in esercizi/input_8000.csv
5. Lo script, tramite query, dovrà restituire quante "Subscription Date" sono dell'anno "2020"

## SQLite
6. Produrre uno script che esegua il load del CSV in esercizi/input_10000.csv
7. Lo script, tramite query, dovrà restituire quanti link in Website sono in "https://"

## DuckDB
8. Produrre uno script che esegua il load del CSV in esercizi/input_2000.csv
9. Lo script, tramite query, dovrà restituire quanti link in Website sono del dominio di primo livello "info"

## Nota
Gli script dovranno produrre i log dei vari step di esecuzione e il tempo totale della stessa.
